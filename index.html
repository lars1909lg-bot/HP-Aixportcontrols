The Problem

Export Control Is Broken

⚠️

Manual Classification

Goods classification still lives in Excel sheets and PDF lists. Technical criteria depend on individual expertise. One mistake can trigger penalties of up to EUR 40 million.

🔌

Disconnected Tools

Sanctions screening, dual-use checks, end-use verification, embargo validation — scattered across multiple systems. No unified workflow. No audit trail. No oversight.

🕑

Days Instead of Minutes

A single export case takes 2–5 business days. Approvals bounce between email chains, Word documents, and spreadsheets. When authorities ask, it takes hours to reconstruct a case.

The Solution

One Platform. Full Control.

AIxportControls replaces today's fragmented tool landscape with a single, AI-assisted compliance workflow — from data ingestion to a fully documented, audit-ready case file. One platform instead of many. Automated. Traceable. Human-in-the-loop.

01

AI-Powered Classification

Automated goods classification against EU Dual-Use, Ammunition List, CCL, ECCN — with AI assistance hints and always up-to-date control lists.

02

Real-Time Sanctions Screening

Continuous screening against EU Consolidated List, OFAC SDN, UK HMT and UN lists. Auto-updated. No manual downloads. No stale data.

03

Integrated Rule Engine

EU Dual-Use Regulation, AWG/AWV, EAR, ITAR, OFAC — all encoded in a versioned, auditable rule engine. Including catch-all logic and end-use verification.

04

Automatic Case File & Audit Trail

Every decision, every source, every timestamp — compiled into a signed PDF case file. Four-eyes principle enforced. Audit-ready by design.

05

ERP Integration & SaaS Deployment

SAP GTS / S/4HANA connector, CSV/XML batch import, REST API. Industry-agnostic. Multi-tenant. Hosted in Germany.
